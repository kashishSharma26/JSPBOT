# JSPBOT - Flask-based Chatbot Solution

JSPBOT is an advanced chatbot solution tailored specifically for Jindal Steel and Power Limited (JSPL), designed to streamline information retrieval processes and enhance user interaction within the organization.

## Key Features

- **Seamless Integration**: JSPBOT seamlessly integrates with Google Chats, providing users with a familiar environment to engage with the chatbot.
- **Information Retrieval**: Users can query JSPBOT to retrieve information from MySQL databases, enhancing efficiency and productivity.
- **API Fabrication**: The Flask framework is utilized to create APIs, allowing JSPBOT to generate JSON files as output for user queries.
- **Modular Architecture**: JSPBOT's modular architecture facilitates code organization and scalability, ensuring flexibility and adaptability to evolving requirements.

## System Requirements

### Software
- Operating System: Windows 11
- Coding Language: Python, HTML, CSS, JavaScript
- Database: SQL Server 2008

### Hardware
- System: HP Pavilion 14
- RAM: 16 GB
- Storage: 512 GB SSD

## Technology Stack

- **Python**: Backend programming language for JSPBOT, offering versatility and robustness.
- **Flask**: Web application framework used to create APIs and facilitate seamless integration.
- **MySQL**: Database management system employed for storing and retrieving data.
- **Google Chats API**: Integrated with JSPBOT to enable direct user engagement within the Google Chats interface.

## Usage

1. Clone the repository.
2. Install the necessary dependencies.
3. Configure the database settings.
4. Run the Flask application.
5. Access JSPBOT through Google Chats.
